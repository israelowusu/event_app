# event_app

A Go-based application for managing events.

## Features

- Issue tracking
- Project boards
- Downloadable releases
- Wiki documentation

## Getting Started

1. **Clone the repository**
   ```sh
   git clone https://github.com/israelowusu/event_app.git
   ```

2. **Build and run**
   - Ensure you have Go installed.
   - Run the app using:
     ```sh
     go run main.go
     ```
